# Growing Sectors Analysis – India Employment & Unemployment (2023-2030)

## 📌 Overview
This project analyzes official government data from **Ministry of Statistics and Programme Implementation (MoSPI)** – PLFS Annual Reports (2024-2025) to identify **growing and declining employment sectors** in India. It includes:

- Sector-wise employment share (Agriculture, Manufacturing, Services)
- Detailed industry-wise employment (PLFS 2023-24)
- Unemployment rates by demographic groups (youth, urban, educated)
- Specialized sectors: Engineering, Pharmaceutical, AI/CS
- Future projections up to 2030 using linear regression
- High-growth future sectors (Care Economy, Gig, AI, Green Energy)

## 🛠️ Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn, Plotly)
- Scikit-learn (Linear Regression for forecasting)
- Official data sources: MoSPI, NITI Aayog, NASSCOM, MeitY

## 📊 Key Findings
- **Agriculture** still largest employer (43%) but declining.
- **Services sector** now employs ~45% of workforce.
- **Youth unemployment (urban)** at 13.6% – alarmingly high.
- **Educated unemployment** (6.5%) double national average.
- **AI & CS sector** grows 18.5% annually but has 7.5% unemployment (skill gap).
- **Care Economy** to create 60 million jobs by 2030.
